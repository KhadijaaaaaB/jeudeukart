# JeuDeuKart

## Description

jeudeukart est un jeu de cartes - **FLIP7** - implémenté en Node.js. Le projet est développé dans le cadre de ELP.

## Règles du jeu

Les règles du jeu sont disponibles dans le document officiel :
- [Règles du jeu](https://github.com/sfrenot/javascript/blob/master/projet4/Regles-Flp-7-vf.pdf)

## Installation

### Cloner le projet

Pour cloner le projet en utilisant HTTPS :

```bash
git clone https://github.com/your-username/jeudeukart.git && cd jeudeukart
```

### Installation des dépendances

Une fois le projet cloné, installez les dépendances npm :

```bash
npm install
```

## Utilisation

### Lancement du jeu

Pour lancer le jeu, exécutez la commande suivante :

```bash
npm run main
```

### Autres commandes

- `npm run test` : Lance les tests du projet
- `npm run lint` : Vérifie la qualité du code
